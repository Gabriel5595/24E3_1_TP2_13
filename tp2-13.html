1. O que significa o conceito de Mobile-first no desenvolvimento web?

  • A) Criar layouts apenas para dispositivos móveis.
  • B) Começar o design pelos dispositivos móveis e adaptá-lo para telas maiores.
  • C) Desenvolver apenas para desktops.
  • D) Usar somente media queries para dispositivos móveis.

Resposta correta: B) Começar o design pelos dispositivos móveis e adaptá-lo para telas maiores.

Justificativa: O conceito Mobile-first visa criar uma experiência otimizada para telas pequenas antes de expandir para telas maiores, garantindo melhor usabilidade.

2. O que é uma media query no CSS?

  • A) Um comando para aplicar estilos a dispositivos móveis.
  • B) Um recurso para testar e aplicar estilos diferentes dependendo das características do dispositivo.
  • C) Uma maneira de mudar as cores no CSS.
  • D) Uma técnica para esconder conteúdo na web.

Resposta correta: B) Um recurso para testar e aplicar estilos diferentes dependendo das características do dispositivo.

Justificativa: Media queries permitem aplicar estilos específicos dependendo do tamanho da tela ou de outras características do dispositivo.

3. Qual é a unidade recomendada para layouts fluidos?

  • A) px
  • B) em
  • C) rem
  • D) %

Resposta correta: D) %.

Justificativa: Unidades percentuais são recomendadas para layouts fluidos, pois ajustam o tamanho de acordo com a largura da tela, tornando o layout mais responsivo.

4. Qual o mínimo de tamanho de fonte recomendado para garantir legibilidade em dispositivos móveis?

  • A) 12px
  • B) 10px
  • C) 16px
  • D) 8px

Resposta correta: B) 10px.

Justificativa: O tamanho mínimo de 10px garante que o texto seja legível mesmo em dispositivos móveis com telas pequenas.

5. Para que serve o atributo viewport nas páginas web?

  • A) Para controlar a largura e o escalonamento da página em diferentes dispositivos.
  • B) Para mudar a cor do background em dispositivos móveis.
  • C) Para esconder elementos em dispositivos menores.
  • D) Para criar animações responsivas.

Resposta correta: A) Para controlar a largura e o escalonamento da página em diferentes dispositivos.

Justificativa: O atributo viewport é essencial para garantir que a página seja exibida corretamente em dispositivos com diferentes tamanhos de tela, ajustando a escala e a largura da visualização.
